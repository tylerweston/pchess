About pchess

pchess was written because I wanted to learn about web-development, and I wanted to make a fun project while I did it.

The server is written in Python using the Flask microframework. It serves up websites that are written using the Jinja templating engine, and use Bootstrap as well to make them look and act nicely. Communication between the front and back-end is done using SocketIO, and in particular the Flask-SocketIO library written by Miguel Grinberg.

The database that stores the games is a PostgreSQL database. The clock is handled by Celery and Redis.

The whole thing is wrapped up nicely using some Docker containers and orchestrated using Docker Compose.
